## Basics

Drivers are being made using Scala programming language. Scala communicates to CoreMidi through JNI library written in Objective C.

### Tools

You'll need:

* OS X Lion
* Midi devices
* Some tools
    * Xcode 4.1
    * IntelliJ Idea, Community Edition is just fine
    * Terminal
